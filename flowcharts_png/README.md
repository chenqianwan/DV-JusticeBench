## 流程图说明

### 1. 系统架构流程图
展示了系统的主要功能模块和用户操作流程，包括：
- 案例管理模块
- AI分析模块
- 分析结果模块

### 2. 案例管理流程
展示了案例管理的两种方式：
- 手动添加案例
- Excel批量导入

### 3. AI分析流程（单个）
展示了单个案例的AI分析流程：
- 问题生成（可选）
- AI分析
- 对比分析（如有法官判决）

### 4. AI分析流程（批量）
展示了批量分析的并发处理流程：
- 任务创建和进度跟踪
- 并发处理
- 实时进度更新

### 5. 结果导出流程
展示了分析结果的导出流程：
- 结果查看
- Excel文件生成
- 文件下载

### 6. 数据流图
展示了系统中数据的流转过程：
- 用户输入 → Web前端 → Flask API
- 数据存储和处理
- 结果导出

## 生成说明

这些PNG图片是从 `web功能流程图.md` 文件中的Mermaid图表自动生成的。

生成工具：`@mermaid-js/mermaid-cli`
生成脚本：`generate_flowchart_png.py`

如需重新生成，运行：
```bash
python3 generate_flowchart_png.py
```
