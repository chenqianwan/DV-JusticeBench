# 打包说明

## 快速打包

### 1. 安装依赖
```bash
pip install -r requirements.txt
```

### 2. 执行打包
```bash
./build.sh
```

或者手动执行：
```bash
pyinstaller pyinstaller.spec
```

### 3. 打包结果
打包完成后，可执行文件位于：
- **macOS/Linux**: `dist/法律AI研究平台/法律AI研究平台`
- **Windows**: `dist/法律AI研究平台/法律AI研究平台.exe`

## 运行打包后的程序

### 方式一：直接运行（会提示输入API密钥）
```bash
./dist/法律AI研究平台/法律AI研究平台
```

程序启动时会提示：
```
============================================================
法律AI研究平台
============================================================

请配置DeepSeek API密钥
提示：您也可以设置环境变量 DEEPSEEK_API_KEY 来避免每次输入
------------------------------------------------------------
请输入DeepSeek API密钥（输入将被隐藏）: 
```

### 方式二：设置环境变量后运行
```bash
export DEEPSEEK_API_KEY=your_api_key_here
./dist/法律AI研究平台/法律AI研究平台
```

## 打包配置说明

- **入口文件**: `run.py` - 支持交互式输入API密钥
- **包含文件**: templates/, static/, data/ 目录
- **控制台**: 启用（用于输入API密钥）
- **单文件模式**: 否（便于调试和查看日志）

## 注意事项

1. **首次运行**: 程序会提示输入DeepSeek API密钥
2. **环境变量**: 可以设置 `DEEPSEEK_API_KEY` 环境变量避免每次输入
3. **数据目录**: 程序会在运行目录创建 `data/` 目录存储案例和结果
4. **端口**: 默认使用 5001 端口，可在 `run.py` 中修改

## 开发模式运行

如果不想打包，可以直接运行：
```bash
python run.py
```

这会启动应用并提示输入API密钥。
